# LocDT8 Practice NFT Smart contract

## Features

- Minting NFT
- Sell & Buy NFT
- ✨Front-end ✨

## Usage

- Deploy contract to Rinkeby
- Run front-end NextJS

```shell
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat node
node scripts/sample-script.js
npx hardhat help
```
